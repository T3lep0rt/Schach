Main erstellt und started ein neues spiel
Spiel verwaltet das spiel und Erstellt einen Spieler, eine AI und ein Spielbrett welches von Spiel initialisiert wird
Spielbrett kann auf schach prüfen, das schachbrett übergeben und initialisieren
Spieler kann einen zug tätigen (durch user input), dieser Zug wird von einem neu erstellten Zughandler geprüft.
Zughandler kann die übergebenen züge überprüfen und gibt true zurück wenn der Zug ausgeführt werden kann.
GUI2D visualisiert alles und besitzt eine liste aller züge.
